This project was made as my final project for my Mobile Application class at Farmingdale University. The project was built using Android Studio, the Kotlin coding language, and used a ROOM database.
This Project was made with the Assistance of CHATGPT AI.

Author: Carlos Gonzalez
Made: 12/8/2025

DESCRIPTION:
This game is a Zelda-style puzzle coding game where the player programs the hero with commands in order to reach the goal, which is the treasure the hero is after.

PROJECT INCLUDES:
- A Custom level maker(Currently still in development).
- Drag and drop Commands.
- A Function maker where users can create their own functions with iterations.
- An Attack input to defeat monsters in your way.
- Dangerous floor traps like Water and Pitfall.
- Persistent data storage using ROOM.
- Child progress tracking using ROOM with filtering and attempt histories.
- Pixelated fonts and visuals that give the game a Zelda-like theme.

GAME FEATURES:
- a sliding movement system with walls, pitfalls, water, monsters, and buttons.
- An attack system with a monster death animation
- Real-time IF block placements done by the player.
- A Function creator with loops and a Gem drag and drop system
- Result dialogs that allow the user to continue to the next level, restart, or exit to the level select screen

TECHNICAL SPECIFICATIONS:
- used Jetpack Compose UI for the entire app
- ROOM database for storing child stats and attempts
- Drag and Drop API
- Coroutine-based animations and movement engine
